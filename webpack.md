# vkui

> A Component Library for Vue.js

## Build Setup

``` bash
# 安装依赖关系
npm install

# 本地开发环境
npm run dev

# 打包完整版
npm run build

# 打包组件
npm dist

# 打包样式文件
build:theme

# 执行发布
npm publish

```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
